Noj_submitter

Introduce

Noj_submitter is a submiiter for noj.cn.

你可以通过Noj_submitter向noj.cn提交代码，或者查看题目。

Usage

输入相关信息->选择语言->提交代码

参数信息：

  • problem: ID of the problem
  • file: 文件路径
Warning: 因为submitter可能不会检查填入信息的正确性，因此你需要确保输入正确的参数

Run .exe in dict

About txt/gif file

  • language.txtr: 存储提交的语言, 1->C++, 2->G++
  • num_to_show_model.txt: 打开Algorithm后显示的数量
  • password.txt: 记录用户信息(暗文)
  • img.gif: 封面图片, 可以用一个大小相同(700x200)的gif代替(文件名保持一致)

FAQ

  • time out: 与noj.cn连接超时
  • problem not find: 1.输入正确的题号; 2.noj.cn炸了
